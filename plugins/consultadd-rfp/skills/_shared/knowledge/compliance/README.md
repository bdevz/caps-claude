# Compliance — Standard Terms & Attestations

Reusable approved language for RFP attestations, MSA standard terms, and compliance representations. Skills inject these verbatim — never reword.

## Files in this directory

- `standard-msa-terms.md` — approved boilerplate for prime contract / MSA negotiations
- `attestations/` — one file per common attestation (SAM.gov registration, EEO, drug-free workplace, debarment, lobbying, etc.)
- `data-handling.md` — approved language for data security, FedRAMP/FISMA references, breach notification commitments
- `subcontracting-plan.md` — small-business subcontracting plan template

> Critical: every file in this directory is reviewed by Legal annually. Skills MUST NOT modify. Updates go through Legal review + PR.
